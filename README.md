# Spotify Music Data Analysis

## Project Overview

This project analyzes Spotify music data to understand patterns in song popularity, audio features, and genre distribution. Using Python data analysis tools, the dataset is explored to uncover insights into musical characteristics such as danceability, energy, tempo, and loudness.

The analysis aims to identify trends that influence song popularity and the relationships between different audio attributes.

---

## Objectives

* Analyze the distribution of song popularity
* Explore relationships between audio features such as danceability and energy
* Identify the most common music genres
* Understand correlations between different musical attributes
* Generate insights from Spotify music data

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook (Anaconda)

---

## Dataset

The dataset contains Spotify track information including:

* Track name
* Artist
* Genre
* Popularity score
* Danceability
* Energy
* Loudness
* Tempo
* Acousticness
* Instrumentalness

The dataset is commonly available through public datasets used for music data analysis.

---

## Analysis Performed

* Data loading and cleaning
* Exploratory Data Analysis (EDA)
* Popularity distribution analysis
* Genre distribution analysis
* Audio feature correlation analysis
* Data visualization of musical attributes

---

## Key Insights

* Songs with higher energy often have higher loudness levels.
* Danceability and energy show a moderate relationship.
* Certain genres dominate the Spotify music catalog.
* Song popularity varies widely across different tracks.
